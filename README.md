# Novel Site Frontend

基于 Vue 3 + TypeScript + Vite 的小说网站前端项目

## 技术栈

- Vue 3
- TypeScript
- Vite
- Pinia
- Axios
- TailwindCSS
- Vue Router

## 项目结构

```bash
novel-site/
├── package.json
├── vite.config.ts
├── src/
│   ├── App.vue
│   ├── main.ts
│   ├── components/
│   ├── views/
│   ├── stores/
│   ├── api/
│   ├── types/
│   └── router/
└── public/
```
